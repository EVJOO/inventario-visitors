🔄 v1 → v2 변경 요약

  • 선 정리 — 2.5px 잉크 테두리 + 4px 하드 섀도우 전부 제거. 무테 카드 + 0 8px 24px rgba(28,34,48,.06) 부드러운 그림자로 통일.
  • 라운드 키움 — 카드 16→18px, 큰 카드 24px, 칩/탭 999px·22px.
  • 스탬프 랠리 제거 — 도장 격자·-8° 회전 그래픽 삭제. 대신 방문 진행 카드(큰 숫자 5/12 + 진행 바) + 찜/방문 통계 미니 2개로 단순화.
  • 부스 카드 재배치 — 좌측 색 스트라이프 → 둥근 사각 썸네일(카테고리 소프트 톤 배경 + 부스코드)로 변경. 카테고리 색이 점이 아니라 면으로 들어와 인지가 명확해짐.
  • 탭바 — 흰 막대 → 둥근 pill 바로. 활성 탭은 잉크 배경 + 흰 글씨로 강한 대비.
  • 찜 / 방문 기능 완전 분리 ✨ v3 업데이트 — 찜은 단순 마킹(별표). 방문은 전체 115개 부스 각각의 독립 상태(방문 전/후).
    진행 카드 기준 변경: 찜한 부스 중 N% 방문 → 전체 115개 중 18곳 방문 (16%).
  • 단일 페이지 구조로 통합 ✨ v4 업데이트
    • 부스 탭(②) · 내 리스트 탭(③) 제거 — 모든 기능을 메인 페이지 하나로 통합. 하단 탭바도 제거.
    • 방문 진행율 카드 상단 이동 — 메타 카드 바로 아래에 18/115 · 16% 완료 풀폭 카드.
    • 인덱스에 검색 + 필터 + 토글 통합
      • 🔎 부스 검색 input (이름 · 부스코드)
      • 필터 칩: 전체 / ⭐ 찜만 / 미방문 / 메모 있음
      • 탭: 알파벳순 ↔ 카테고리별 (같은 토글·검색 공유)
    • 행 강화 — 행 크기 키움(높이 ~56px). 각 행에 3개 토글 + PLATZ 배지:
      • ⭐ 찜: 비활성=외곽선 별 / 활성=옐로우 배경 + 채워진 별
      • ✓ 방문: 비활성=점선 빈 원 / 활성=그린 배경 + 체크 (도장 컨셉)
      • 📝 메모: 비활성=외곽선 메모 / 활성=시안 배경 + 채움
    • 행 클릭 → 상세 모달 — 메모 내용 확인·편집·찜·방문 모두 가능.
    • 방문 아이콘 디폴트 변경: 별 도장(A안) → 체크 도장(B안). 찜의 별과 시각 구분 + "체크 표시 활성화" 요구에 부합.
    • 데스크탑 그리드: 메인 페이지 + 상세 모달 = 2열 (모바일에선 단일 컬럼).
  • 카테고리 색 → 원본 PDF 추출값으로 교체 ✨ v3.4 업데이트
    • 원본 PDF의 부스 코드 텍스트를 픽셀 단위 샘플링해 8개 카테고리 색 추출.
    • 이전 시안의 임의 컬러(예: SMALL=#FF5C8A, PAPER=#57C84D)를 원본 값(SMALL=#F4B4C4, PAPER=#00B8F0)으로 전면 교체.
    • 특히 PAPER ↔ DAILY가 시안 ↔ 그린 매핑(이전 시안은 반대)이 원본 의도였음 — 정정.
    • 모든 사용처(부스 카드 썸네일, 카테고리 라벨, 인덱스 행 점, 미니 리스트, 진행 카드, 비교 박스)가 CSS 변수 한 줄 교체로 자동 반영.
    • 가독성 보완: 옐로우(#DCC400)·라이트 시안(#80DCF4)·라임 그린(#4CF448)·라이트 핑크(#F4B4C4)는 원본이 회색 배경 디자인이라 흰 배경에서 옅게 보임. 부스 카드는 소프트 톤 배경 + 풀 컬러 텍스트로 가독성 확보.
  • 지도 페이지 다듬기 ✨ v3.3 업데이트
    • 헤더 중복 제거 — 지도 페이지에서만 .app-head(작은 wordmark·날짜) 제거하고 핑크 메타 카드만 남김. 정보 노출 1회로 정리.
    • 인덱스 행 포맷 통일 — 카테고리별 2열 그리드 → 알파벳순과 동일한 1열 행 포맷(idx-row)으로 통합. UX 일관성 ↑.
    • PLATZ 위치 배지 — 모든 행 우측에 P1 · P2 · L 색 구분 배지. 알파벳·카테고리 양쪽 동일.
    • 매핑 기준: A·B·C·D = PLATZ 2, E·F·G·H·I·J·K·L = PLATZ 1, M = LOUNGE(공용 라운지).
  • 지도 페이지 재구성 ✨ v3.2 업데이트
    • 핑크 메타 카드 대폭 축소 — 한 줄짜리(타이틀 + 장소·날짜 + D-DAY 뱃지). 페이지 상단 공간 절약.
    • 배치도 강조 — 흰 카드(라운드 20px) + 부드러운 그림자로 시각적 무게 ↑. 카테고리 범례 카드 제거(인덱스에서 대체).
    • PLATZ HALL 1 / HALL 2 분리 — 원본 PDF에서 각각 잘라서 별도 카드로. 카드별 가로 스크롤(폭 780px). 푸터에 해당 홀 구역(예: A·B·C·D + LOUNGE) 표기.
    • 부스 인덱스 신설 — 배치도 아래 탭 컨트롤(알파벳순 ↔ 카테고리별). 알파벳순은 A∼L·M 그룹 헤더 + 행. 카테고리별은 8개 카테고리 칩 + 2열 행 그리드. 카테고리 점 색으로 색맥락 유지.
    • 두 탭 모두 시안에 노출(아래쪽이 카테고리별 미리보기) — 실제 빌드 때는 한 번에 한 탭만 보임.
  • 방문 아이콘 → 스탬프 컨셉 ✨ v3.1 업데이트 — "도장 누르기 전/후" 메타포. 비활성: 점선 빈 자리(아직 도장 안 찍힘). 활성: 그린 채움 + 살짝 기울어진(-10°) 도장 + 잉크 번짐 점.
    기본 적용: A안 별 도장. 다른 안: B안 체크 도장, C안 한글 "방" 도장. 시안 하단 비교 박스에서 직접 보고 선택.
    UI 텍스트는 "방문 전 / 방문 체크"로 그대로 유지(스탬프 메타포는 아이콘 비주얼에만).
  • 폰트 — 하이브리드 ✨ v2.1 업데이트 — 본문/카드/라벨은 Pretendard로 가독성 유지, 큰 타이틀에만 꾸불림체를 액센트로 적용 → 귀엽고 키치한 인상.
    적용: inventario 워드마크 · 티켓 타이틀 · 방문 진행 큰 숫자 5/12 · 상세 시트 부스명 · 섹션 타이틀(찜한 부스/방문 완료).
    제외: 부스 카드 한글명(14px) · 칩 · 버튼 · 본문 — 가독성 우선.
  • 도트 패턴 배경 제거 — 깔끔한 #F4F6FA 단색 배경.

🧐 확정해야 하는 결정 포인트 (v2)

  • 티켓 히어로 — cyan 카드에 흩어진 원 데코 + 펀치홀 느낌의 이중 보더 원. 너무 가벼우면 잉크 데코를 더 넣을 수 있어요. 너무 화려하면 단순 카드로.
  • 방문 진행 카드 — 큰 숫자(5/12) + 한 줄 진행 바. 다른 안: 원형 게이지, 도넛 차트, 일러스트 + 숫자.
  • 토글 — 카드 우측 별/체크 두 개를 가로 정렬. 한 손 엄지로 닿기 좋아요. 더 큰 사이즈가 좋으면 키움.
  • 탭 아이콘 — 지금은 이모지(🗺️🏪📋). 라인 SVG 아이콘으로 교체하면 더 깔끔해져요. (레퍼런스 4번도 라인 아이콘)
  • 꾸불림체 완전 제거 OK? — prompt 사양엔 헤드라인용으로 적혀있었어요. 이번 v2에선 무드 통일을 위해 뺐는데, 브랜드 워드마크 하나에만 살리는 안도 가능합니다.
